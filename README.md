# native-script-map
Low level plugin for NodeJS that implements a structure for single string concat in c++. Don not ask why ;]. Cousin of https://github.com/sp3c1/native-string-map. 

# Test
```
mocha test\index.js
```
or
```
node node_modules/.bin/mocha test/index.js
```

# Build
```
node-gyp configure
node-gyp build
```
